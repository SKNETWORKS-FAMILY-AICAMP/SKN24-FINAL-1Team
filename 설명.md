# 실행 방법
## 1. 백엔드 실행
cd backend/hpm
pip install -r ../requirements.txt
python manage.py migrate
python manage.py runserver 0.0.0.0:8000

## 2. 프론트엔드 실행
cd frontend/hpm
npm install
npm run dev
